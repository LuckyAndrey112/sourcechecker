# Source Checker
Интерфейс для проверки наличия алертов для log source и log source type в IBM Qradar и VictoriaMetrics.
Начиная с версии 0.4.4 доступны проверки в MP SIEM

# Версия
Текущая версия - 0.4.5

# Переменные окружения
- PROM_URL
- PROM_USERNAME
- PROM_PASSWORD
- SEC_TOKEN
- QRADAR_HOST
- TTC_QRADAR_HOST
- TTC_SEC_TOKEN
- GITLAB_URL
- GITLAB_TOKEN
- SECRET_KEY
- SECRET_KEY_MP
